Services

I am eager to explore other opportunities in web development where my technical skills can provide service and support for an organization that would grant me the opportunity.

Most of my experience is in web development and web design.

The tools I use to provide web development are Visual Studio 2008, SQL Server 2008, SQL Server Management Studio, and Microsoft Windows Server 2008. Visual Studio 2008 is used for developing the web application graphical user interface (GUI) or some would call it the "front end". I use SQL Server 2008 to create and maintain the databases that are used for the web applications, which would be called the "back end" of the web application. I familiar with SQL Server Management Studio being used to create and administer the functions of the database such as the stored procedures, views, triggers, and queries.

The tools I use to provide web design are Adobe Dreamweaver CS4, Adobe Photoshop CS4, Adobe Illustrator CS4, and Adobe Flash Professional CS4. I use Dreamweaver as a development environment to combine text and images to create web pages which produces a website. I used Adobe Photoshop for digital imaging which is needed for the images of a web pages background or visual displays and also for any pictures or images that will be displayed as the content of the website. I use Illustrator to create logos and for digital imaging that may require the use of vectors to manipulate an image. I use Adobe Flash Professional to produce animation and multimedia content for the web as well.
